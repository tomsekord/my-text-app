# My Text App

A simple web app that lets you type text and customize its color and background in real time.

## Live Demo
https://tomsekord.github.io/my-text-app

## Features
- Enter custom text
- Pick text color
- Pick background color
- Toggle between edit and preview view

## Built With
- HTML
- CSS
- JavaScript

## How to run locally
Open `index.html` in any browser.
